## ②課題内容（どんな作品か）
- 「料理するのがめんどくさい…」「料理を続けようとしても続かない…」そんな人に向けた、ズボラ飯特化のレシピSNS。ズボラで美味しいが評価されるSNS。
- 自分でレシピ登録できるとともに、他の人のレシピをお気に入り登録することも可能。

## ③アプリのデプロイURL

## ⑤工夫した点・こだわった点
- レシピ一覧をカード形式にして見やすくした点
- 人それぞれ「料理がしやすいスタイル」も「楽と感じるポイント」も違ったりするので、「使用する道具」「実働時間」「洗い物レベル」「料理スタイル派閥」など、様々な判断基準を入れた点。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 画像の取り扱いが難しく、かなりAI頼りになってしまった。
- 検索機能やレシピの絞り込みができるようにして、より使いやすいUXを目指したい。
- レシピを見た人がズボラ度や美味しさなどを評価できる機能や、コメントを残せる機能も実装してみたい。
